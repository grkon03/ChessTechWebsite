mysql.php
---------------
classes:
    - Members
        - データベースの Members テーブルの情報を格納する型としてのクラス
    - Schedules
        - データベースの Schedules テーブルの情報を格納する型としてのクラス
    - JoinableDay
        - データベースの JoinableDays テーブルの情報を格納する型としてのクラス
    - MYSQL_UTIL
        - データベースを操作するためのAPIを集めたクラス
        variables:
            - $pdo
                - PDO オブジェクト
        functions:
            - Members テーブル関連
                - CreateMember(Member $mem)
                    - $mem の情報に基づいて Members テーブルにデータを追加
                    - 成功したら true, 失敗したら false を返す
                - UpdateMember(Member $mem)
                    - $mem の情報に基づいて Members テーブルのデータを編集
                    - 編集されるデータは id が $mem->id であるデータ
                    - 成功したら true, 失敗したら false を返す
                - AuthenticateMember($id, $pass)
                    - id が $id, pass が $pass であるようなデータが Members テーブルに存在するか調べる
                    - 存在したら true, 存在しなければ false を返す
                - GetMember($id)
                    - id が $id であるようなデータを Members テーブルから探す
                    - 存在すれば Members 型変数としてデータを返す、存在しなければ null を返す
                - GetAllMembers()
                    - Members テーブルのデータを全て取得する
                    - Member 型変数の配列としてデータを返す、存在しなければ null を返す
                - DeleteMember($id)
                    - id が $id であるようなデータを Members テーブルから消去する
                    - 成功したら true, 失敗したら false を返す
            - Schedules テーブル関連
                - CreateSchedule(Schedule $sch)
                    - $sch の情報に基づいて Schedules テーブルにデータを追加
                    - $sch->id は指定しなくて良い
                    - 成功したら true, 失敗したら false を返す
                - UpdateSchedule(Schedule $sch)
                    - $sch の情報に基づいて Schedules テーブルのデータを編集
                    - 編集されるデータは id が $sch->id であるデータ
                    - 成功したらこのデータの id を返す, 失敗したら false を返す
                - GetSchedule(int $id)
                    - id が $id であるようなデータを Schedules テーブルから探す
                    - 存在すれば Schedule 型変数としてデータを返す、存在しなければ null を返す
                - GetSchedulesBetween(DateTime $d_from, DateTime $d_to)
                    - $d_from から $d_to の期間内にあるスケジュールを Schedules テーブルから探す
                    - $d_from, $d_to は共に日にちの指定であって、時間帯は指定不可
                    - 存在すれば Schedule 型変数としてデータを返す、存在しなければ null を返す
                - GetAllScheducles()
                    - Schedules テーブルのデータを全て取得する
                    - Schedule 型変数の配列としてデータを返す、存在しなければ null を返す
                - DeleteSchedule(int $id)
                    - id が $id であるようなデータを Schedules テーブルから消去する
                    - 成功したら  true, 失敗したら false を返す
            - JoinableDays テーブル関連
                - CreateJoinableDays(JoinableDay $joi)
                    - $joi の情報に基づいて JoinableDays テーブルにデータを追加
                    - $joi->date のスケジュールが存在したら false を返す
                    - 成功したら true, 失敗したら false を返す
                - UpdateJoinableDays(JoinableDay $joi)
                    - $joi の情報に基づいて JoinableDays テーブルのデータを編集
                    - 編集されるデータは date が $joi->date であるデータ
                    - 成功したら true, 失敗したら false を返す
                - AddMemberJoinableDay(JoinableDay $add_joi)
                    - date が $add_joi->date であるようなデータを JoinableDays データにメンバーを追加
                    - $add_joi の各項目のメンバーをデータに追加
                    - 成功したら true, 失敗したら false を返す
                - RegistMemberJoinableDay(DateTime $date, string $user_id, int $state)
                    - date が $date であるような JoinableDays のデータにメンバーは追加される
                    - id が $user_id であるようなメンバーを $state に基づいて追加または消去する
                        - $state
                            - 0: メンバーを消去
                            - 1: joinable に追加
                            - 2: maybe_joinable に追加
                            - 3: notjoinable に追加
                    - 成功したら true, 失敗したら false を返す
                - GetJoinableDays(JoinableDay $bind)
                    - 条件が $bind と一致するようなデータを JoinableDays テーブルから探す
                    - JoinableDay 型変数の配列としてデータを返す、存在しなければ null を返す
                - GetAllJoinableDays()
                    - JoinableDays テーブルのデータをすべて取得する
                    - JoinableDays 型変数の配列としてデータを返す、存在しなければ null を返す
                - DeleteJoinableDay(DateTime $date)
                    - date が $date であるようなデータを JoinableDays テーブルから消去する
                    - 成功したら true, 失敗したら false を返す
                - DeleteMemberJoinableDay(DateTime $date, string $user_id)
                    - date が $date であるようなデータからメンバーを消去する
                    - id が $user_id であるようなメンバーが消去される
                    - 成功したら true, 失敗したら false を返す
